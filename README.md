# Employee Management Portal

A frontend-only employee management portal built with HTML, CSS, and vanilla JavaScript.

## Features
- Login flow with LocalStorage-backed authentication simulation
- Dashboard overview with static charts and activity summaries
- Employee directory with search, sorting, filtering, pagination, and modals
- Profile management with inline validation
- Settings with persisted preferences
- Notifications inbox with filter and search
- About page with portal statistics

## Run locally
Open any HTML page directly in a browser, or serve the directory with a simple static server such as `python -m http.server 8000`.
